# Dice Roller Game

A console-based dice rolling game built in Python — my first project, used to practice core programming fundamentals.

## Features
- Roll a virtual die with randomized outcomes
- Tracks roll history and score across a session
- Detects and flags lucky/unlucky streaks

## How to run
1. Clone the repo: `git clone https://github.com/samihasoyshee/python-dice-roller-game.git`
2. Run: `python main.py`

## What I learned
Functions, loops, lists, and conditional logic in Python.
